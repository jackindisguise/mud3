    Function text2dir

    • text2dir(text: DirectionText): DIRECTION

      Converts a text representation to its DIRECTION enum value.

      Accepts both full direction names (e.g., "north", "northeast") and abbreviated forms (e.g., "n", "ne"). The search is case-insensitive for convenience.

      Parameters

      • text: DirectionText

        The direction text to convert (full or abbreviated)

      Returns DIRECTION

      The DIRECTION enum value, or undefined if not found

      Example

      Full direction names:

      text2dir("north"); // DIRECTION.NORTH
      text2dir("northeast"); // DIRECTION.NORTHEAST
      text2dir("up"); // DIRECTION.UP

      Example

      Abbreviated direction names:

      text2dir("n"); // DIRECTION.NORTH
      text2dir("ne"); // DIRECTION.NORTHEAST
      text2dir("u"); // DIRECTION.UP

      Example

      Use in command parsing:

      function processMove(input: string) {
        const dir = text2dir(input);
        if (dir !== undefined) {
          player.step(dir);
        } else {
          console.log("Invalid direction");
        }
      }
