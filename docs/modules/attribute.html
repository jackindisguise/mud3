    Module attribute

    Attribute system for mobs, including primary attributes, secondary attributes, resource capacities, and helper functions for calculating and manipulating them.

    Primary attributes (strength, agility, intelligence) are the base stats that determine secondary attributes and resource capacities. Secondary attributes (attackPower, defense, critRate, etc.) are derived from primary attributes. Resource capacities (maxHealth, maxMana) are calculated from base values, secondary attributes, and bonuses.
