    • generateMinimapFromSteps(
          room: Room,
          mob: Mob,
          size: number,
      ): string[] | undefined

      Generates a minimap showing rooms around the current room using room.getStep(). This version respects portals and room links by stepping from room to room rather than using coordinate-based lookups. Displays a grid of rooms with the current room marked. Dense rooms block vision of rooms behind them.

      Parameters

      • room: Room

        The center room for the minimap

      • mob: Mob

        The mob viewing the minimap (for checking exits)

      • size: number

        The number of tiles to show in each direction (size 1 = 3x3, size 2 = 5x5)

      Returns string[] | undefined

      A string representation of the minimap, or undefined if no dungeon
