Help & Tips

Editing Archetypes

  • Select a race or job from the sidebar to edit it
  • Edit the archetype properties using the form fields
  • Click "Save" to save your changes
  • Click "Delete" to remove an archetype
  • Use "Add Race" or "Add Job" to create new archetypes
  • Select abilities from the dropdown to add them to the archetype
  • Edit ability levels directly in the ability boxes
  • Use "+ Add Passive" to add passive abilities
